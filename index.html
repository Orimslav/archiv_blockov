Archivujte e-kasa bločky jednoducho

Naskenujte QR kód bločku a aplikácia ho uloží, zaradí podľa dátumu, rozpozná predajcu aj položky a pripraví reporty, exporty a prehľady.

Stiahnuť aplikáciu Windows aj Linux

Windows (.exe) aj Linux (AppImage) · bez inštalácie · 🔒 všetko lokálne, žiadny cloud

Čo aplikácia dokáže

Všetko, čo potrebujete na poriadok v bločkoch — pre domácnosť aj firmu.

📷

Skenovanie QR

USB 2D (QR) čítačka (ako klávesnica) alebo ručné zadanie UID. Online aj offline bločky vrátane položiek.

📁

Profily a kategórie

Viacero profilov (domácnosť, firma). Kategórie na úrovni položiek s učením podľa predajcu.

📅

Zakladanie podľa dátumu

Bloček sa zaradí podľa dátumu z QR kódu — bez ohľadu na to, kedy ho naskenujete.

🔎

Hľadanie položiek

Vyhľadajte položku naprieč bločkami, sledujte spotrebu a vývoj ceny v čase. Aliasy zlúčia varianty názvov.

🧾

Verný doklad

Každý uložený bloček znova zobrazíte ako tlačiteľnú kópiu s QR kódom — aj offline.

📊

Exporty a dashboard

PDF prehľady, súhrny kategórií, DPH podklad, Excel/CSV a grafy výdavkov po mesiacoch.

💾

Záloha a obnova

Bezpečná záloha a obnova databázy s validáciou a automatickou zálohou pred prepisom.

🔐

Voliteľné heslo

Aplikáciu si môžete zaheslovať (bcrypt) — alebo nechať bez hesla. Rozhodnutie je na vás.

🔒

Všetko lokálne

Bločky aj dáta sa ukladajú výhradne vo vašom počítači (SQLite). Žiadny cloud, nič sa nikam neposiela.

Náhľady

Prehľadné tmavé rozhranie optimalizované pre Full HD.

Zoznam bločkov (domácnosť)
Rozpad DPH (firemný profil)
Dashboard — výdavky po mesiacoch a kategóriách
PDF export — DPH podklad

Ako to funguje

Od naskenovania po hotový archív v štyroch krokoch.

1

Naskenujte

Priložte QR kód bločku k USB skeneru.

2

Skontrolujte

Aplikácia načíta predajcu, dátum, sumu aj položky.

3

Zaraďte

Vyberte kategóriu — bloček sa uloží do správneho obdobia.

4

Vyhodnoťte

Prezerajte reporty, exporty a grafy kedykoľvek.

Stiahnutie a spustenie

Stiahnite hotový súbor z vydaní pre Windows aj Linux, alebo si aplikáciu zostavte zo zdrojového kódu.

📷 Na skenovanie je potrebná USB 2D (QR) čítačka čiarových kódov (bežná 1D laserová nestačí). Bločky sa dajú zadať aj ručne.

🪟

Windows

ArchivBlockov.exe — dvojklik, netreba inštaláciu ani Python.

🐧

Linux

ArchivBlockov-x86_64.AppImage — chmod +x a spustiť. Prenosné, beží na väčšine distribúcií (vyžaduje FUSE).

Ku každému súboru je priložený kontrolný súčet .sha256. Alebo zostavenie zo zdrojového kódu:

:: Windows
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\python main.py

# Linux
python3 -m venv venv
venv/bin/pip install -r requirements.txt
venv/bin/python main.py
